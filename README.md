================================
实验2：DTN 梦境机制 - 让系统自己长出知识
================================

实验目标：验证 DTN 能否从极少量初始规则出发，通过内部逻辑推理和随机组合，生成有意义的、新的规则。

核心问题：能否让一个离散触发网络（DTN）在没有外部输入的情况下，自发地创造出新的知识？

================================
这个实验是干什么的？
================================

想象你给一个系统 5 条简单规则（比如苹果是水果、水果有颜色）。

然后让它自己"做梦"：
- 不断推理、组合
- 自己发现新规则（比如苹果有颜色）
- 不需要外部输入，不需要大模型

这个实验就是在研究：系统能不能自己"长出"知识。

================================
实验内容
================================

初始规则（5 条）：
--------------------------
(苹果, 是, 水果)
(水果, 有, 颜色)
(香蕉, 是, 水果)
(雪, 是, 白色)
(白色, 是, 颜色)

推理机制：
--------------------------
1. 传递性推理（70% 概率）：
   如果 A→B 且 B→C，则 A→C
   例子：苹果→水果，水果→有颜色 → 苹果→有颜色

2. 随机组合（30% 概率）：
   随机拼接两个规则
   可能产生无意义规则，但会被一致性检查过滤

3. 一致性检查：
   不允许矛盾规则（如"苹果有颜色"和"苹果没有颜色"）

期望推导出的规则：
--------------------------
(苹果, 有, 颜色)  ← 通过传递性
(香蕉, 有, 颜色)  ← 通过传递性

================================
实验结论
================================

✓ DTN 成功从初始规则推导出新规则
✓ 传递性推理生成了有意义的规则
✓ 随机组合提供创新性（虽然可能产生无意义规则）
✓ 一致性检查保证知识库的合理性
✓ 证明了 DTN 可以在没有外部输入的情况下，自发地创造新知识

核心价值：
- 绕开"从大模型提取"的难题
- 让智能从内部诞生
- 可解释性强，每条规则都看得懂
- 可控制，推理过程完全透明
